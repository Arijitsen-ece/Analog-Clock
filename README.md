🕰️ Professional Analog Clock

A modern and responsive Analog Clock built using **HTML, CSS, and JavaScript**.
This project displays a real-time working clock with smooth rotating hour, minute, and second hands based on the user's local time zone.

---

## 🚀 Features

- Real-time analog clock
- Smooth hand rotation using JavaScript
- Time zone display
- Clean gradient background
- Responsive and centered layout
- Pure CSS clock design (no images)

---
## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
